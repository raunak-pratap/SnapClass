# 🎓 SnapClass

SnapClass is an AI-powered classroom management system built with Streamlit and Supabase.

## 🚀 Live Demo

🌐 **Live Application:** https://snapclass-masters.streamlit.app/

## ✨ Features

- 👨‍🏫 Teacher Dashboard
- 👨‍🎓 Student Dashboard
- 😊 Face Recognition Attendance
- 🎤 Voice Attendance
- 📱 QR Code Attendance
- 📊 Attendance Analytics
- ☁️ Supabase Database
- 🔐 Secure Authentication

## 🛠 Tech Stack

- Python
- Streamlit
- Supabase
- dlib
- Face Recognition
- Resemblyzer
- Librosa
- Pandas
- NumPy

## Installation

```bash
git clone https://github.com/raunak-pratap/SnapClass.git
cd SnapClass
pip install -r requirements.txt
streamlit run app.py
```
